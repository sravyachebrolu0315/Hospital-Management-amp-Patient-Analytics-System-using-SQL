# Hospital-Management-amp-Patient-Analytics-System-using-SQL

<h3>Problem Statement: Hospital Management & Patient Analytics System </h3>
<p>Hospitals need efficient systems to manage patient records and analyze treatment data.</p>

<h3>Objectives:</h3>
<ul>
<li>Design a database to store patient, doctor, and treatment data</li>
<li>Perform analysis on hospital operations</li>
</ul>

<h3>Database Tables:</h3>
<ul>
<li>Patients (patient_id, name, age, gender)</li>
<li>Doctors (doctor_id, name, specialization)</li>
<li>Appointments (appointment_id, patient_id, doctor_id, date)</li>
<li>Treatments (treatment_id, patient_id, diagnosis, cost)</li>
</ul>

<h3>Key Tasks:</h3>
<ul>
<li>Find most consulted doctors</li>
<li>Calculate total revenue per month</li>
<li>Identify most common diseases</li>
<li>Track patient visit frequency</li>
<li>Analyze doctor performance</li>
</ul>

<h3>Expected Outcome:</h3>
<p>A robust database system that helps in managing hospital data and extracting insights.</p>
